# Python examples

Notion API integrations built with Python and the official [`notion-client`](https://github.com/ramnes/notion-sdk-py) SDK.

## Available examples

- **[intro-to-notion-api](intro-to-notion-api/)**: Four short scripts covering the core operations — adding blocks, creating databases, adding pages, and querying with filters

## Getting started

Each example has its own `README.md` with setup instructions. In general:

1. Create a [Notion integration](https://www.notion.com/my-integrations)
2. Share a test page with your integration
3. Create a virtual environment: `python -m venv .venv && source .venv/bin/activate`
4. Install dependencies: `pip install -r requirements.txt`
5. Copy `example.env` to `.env` and fill in your credentials
6. Run the scripts
